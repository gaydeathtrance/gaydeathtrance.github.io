Add the Image

  1. Find the image that you want to accompany the piece.
  2. You'll have to type the name of the file a few times, so I would rename the image to have a small, memorable name.
  3. Add this image to the folder "images".

Editing the Author Page

A) For Authors Without a Page

With the image all set we're ready to make the page for the new author and the page for their work.
  1. Duplicate the folder "author_name". Name it after the author. Generally I say use all lowercase, but for you and BM I have used capitals and just the initals.
  2. Now it's time to make the author's page which is the "index.html" in their folder.
  3. Open "index.html" and press cmd+F.
  4. Find "Author Name" and replace all with the author's name.
  5. Scroll down to the link. All you need to do is change "Work Name" with the title of the piece.

B) For Authors With a Page

  1. Go to the folder of the author who has a new piece.
  2. Duplicate the file "template.html" file and call it "number.html" where "number" is the next number along.
  3. Open "index.html" and scroll to the links. I have added a template link in a comment.
  4. Copy this link and paste it below the current links. Replace "number.html" with the correct number (the page you just created).
  5. Replace "Work Name" of this new link with the title of the piece.

Adding New Work

  1. Open the file for the new work. If this is the author's first piece, that'll be "1.html". If it's not, you'll be opening your most recent "number.html".
  2. Press cmd+F
    • Search "Author Name" and replace all with the author's name.
    • Search "Work Name" and replace all with the Work's title.
    • Search "imagename.jpg" and replace all with the image's name.
      •   Note: when you replace that make sure you include .jpg with the replace all. If you're using a png then replace the "imagename.jpg" with yourimage.png. You get the idea.
  3. Paste in the work. This goes in between <p class="workBody"> ... insert here ... </p>. (More details below.)
  4. The author's name will already be in the bio, but make sure you add the rest of it after the <b> Name </b>.
  5. Fill in the image credits.

Formatting the work

  • Every new line must have a <br>. Neat trick using atom, if you type "br" and press enter, it'll add the brackets for you.
    • If you want to do that trick with text edit, open "textEdit" and press cmd+F
    • Click the magnifying glass and press "insert pattern" and choose "line break"
    • Copy the line break and paste it into the "replace"
    • Replace all "line breaks" with "<br>line break"
  • To make text bold: surround it by <b>Bold</b> = Bold
  • To make text Italic: surround it by <i>Italic</i> = Italic
  • To make text underline: surround it by <u>Underline</s> = Underline
  • To make text strikethrough: surround it by <s>Strikethrough</s> = Strikethrough
  • To add an emdash, sometimes it's easiest to just replace all double dashes with "&mdash;"

Adding to Front Page

A) Add Piece to Feature

If you want to add the newest piece to the feature then we have to do two things. Move the current piece into the archive and add the newest one to the featured work. First lets put the older piece into the archive.
  1. Open "index.html" in the main folder. Scroll down until you see "COPY THE BELOW LINES".
  2. Copy that part, and paste below my comment "Paste the new archive content below".
  3. There are three lines that you need to change:
    1. The first line replace "AUTHOR/number.html" with the right information for what used to be featured. If you're not sure, that link will be in the currently featured bit. Also replace "imagename.jpg" with the image from the previously feautred piece.
    2. Now you're editing <a href="author/number.html">WORK TITLE</a>, replacing "author/number.html" with the same info you did above, and the "WORK TITLE" with the title.
    3. This is the link to the author page. Edit <a href="author/index.html">AUTHOR NAME</a>. You should replace "author" with the folder name for that author and "AUTHOR NAME" with their full name.
  4. Now we're going to add the page you just made to the featured part. Scroll up until you see "EDIT THIS TO CHANGE FEATURED CONTENT".
  5. Just like above there are three lines to change.
    1. First line: change the work link "author/number.html" and the image link "images/imagename.jpg"
    2. Second line: change the work link "author/number.html"
    3. Third Line change the author link "author/index.html"

B) Add Piece Straight to Archive

If you want to keep the current piece in featured and add the newest piece to the archive then just do this:
  1. Open "index.html" in the main folder. Scroll down until you see "COPY THE BELOW LINES".
  2. Copy that part, and paste below my comment "Paste the new archive content below".
  3. There are three lines that you need to change:
    1. The first line replace "AUTHOR/number.html" with the file you just made. Also replace "imagename.jpg" with the image you just added.
    2. Now you're editing <a href="author/number.html">WORK TITLE</a>, replacing "author/number.html" with the same info you did above, and the "WORK TITLE" with the title.
    3. This is the link to the author page. Edit <a href="author/index.html">AUTHOR NAME</a> replace "author" with the folder name for that author and "AUTHOR NAME" with their full name.
  4. Done!

Check Everything

  1. Open the "index.html" using Chrome
  2. Does everything look ok?
  3. Now try clicking the image, the work title and the author's name
  4. The image and the work title should take you to the piece. Does it?
  5. The author's name should take you to the author's page. Does it?
  6. Author's page should have the new work on it. Does it?
  7. Is the work formatted the way it was sent? Might need to add / remove line breaks to be happy.
